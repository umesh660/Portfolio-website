PROJECTS

  1. Image Classification Model Development [Python, TensorFlow, Keras]

    Developed an image classification model using Python in a Google Colab environment, leveraging TensorFlow and Keras libraries. The project aimed to classify images into "Happy" and "Sad" categories, implementing data preprocessing, model development, training, and evaluation steps. Achieved a 93.5% accuracy on the test set.

    • Data Preprocessing: Validated image extensions and removed unsupported files, ensuring compatibility for model training.
    • Model Development: Built a CNN using Keras, including convolutional, max-pooling, and dense layers for feature extraction and classification.
    • Model Training: Split data into training, validation, and test sets; trained using Adam optimizer and binary cross-entropy loss.
    • Model Evaluation: Assessed precision, recall, and accuracy, with a final accuracy of 93.5% on the test set.
    • Prediction Demonstration: Demonstrated the model's ability to classify new images as "Happy" or "Sad."

  2. Forecasting Economic Indicators and FTSE 100 [Python, ARIMA, Exponential Smoothing]

    This technical report forecasts UK economic indicators and the FTSE 100 index using datasets from the UK Government Office for National Statistics (ONS). The report applies exponential smoothing and ARIMA models to predict trends until December 2024, aiding in strategic decision-making.

    • Data Preparation: Sourced and normalized monthly data from ONS, handling missing values.
    • Forecasting Methods: Applied exponential smoothing and compared it with ARIMA for specific indicators like Average Weekly Earnings (K54D).
    • Multivariate Regression Model: Used multiple economic indicators to predict the FTSE 100 index.
    • Results: Presented forecasts and compared model performance, providing actionable insights for stakeholders.
    • Appendix: Included Python code for data preparation, model development, and validation.
